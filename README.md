# Wikipedia Link Path Game

Navigate from one Wikipedia article to another using only in page links. Each click increases a counter and your path is recorded.

## How to play
1. Open the site
2. Paste a start Wikipedia article link
3. Paste a target Wikipedia article link
4. Select Start game
5. Click only article links inside the page to move toward the target
6. The game ends when the target article loads
